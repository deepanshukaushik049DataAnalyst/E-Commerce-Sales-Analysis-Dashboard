# 📊 E-Commerce Sales Analysis Dashboard

## Overview

This project presents an end-to-end E-Commerce Sales Analysis Dashboard developed using Python and Tableau. The objective is to analyze sales performance, profitability, customer activity, and product trends to generate actionable business insights.

The project includes data cleaning and preprocessing in Python, exploratory data analysis (EDA), and an interactive Tableau dashboard for business performance monitoring.

---

## Business Problem

E-commerce businesses generate large volumes of transactional data every day. Analyzing this data helps stakeholders understand sales trends, identify profitable categories, evaluate regional performance, and make data-driven business decisions.

This dashboard provides a centralized view of key business metrics and performance indicators.

* Total Sales
* Total Profit

### Visualizations

#### Sales Trend

Tracks sales performance over time and identifies growth patterns.

#### Sales by Region

Compares revenue generated across different regions.

#### Sales by Category

Highlights category-wise sales contribution.

#### Profit by Category

Evaluates category profitability and identifies high-performing segments.

#### Top 10 Products

Displays the highest revenue-generating products.

#### State-wise Sales Map

Visualizes geographic sales distribution across states.
